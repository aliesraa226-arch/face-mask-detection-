📌 CNN Models with Live Stream Prediction
📖 Project Overview

This project implements and compares three Convolutional Neural Network (CNN) models on a new dataset.
It also includes a live stream system using a webcam to perform real-time predictions.
***************************************************************************************************************
🎯 Objectives
Train and evaluate 3 CNN models:
ResNet50
InceptionV3
MobileNetV2
Compare their performance (accuracy, speed, efficiency)
Deploy a live camera stream for real-time predictions
***************************************************************************************************************
📂 Dataset
Dataset used: (face mask detection)
Number of classes: (With Mask / Without Mask)
Images were resized to 224 × 224
Data split:
Training set
Validation set
Test set
